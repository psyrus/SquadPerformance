How to contribute

Due to the lack of benchmarking tools integrated into the main squad program, the smoothest solution that I could come up with was using a combination of free software and the "honor system" (I'm trusting the submissions) to establish some decent performance baselines for squad players around the world.

Ideally, if this system garners a few thousand submissions from a range of systems, it will provide a very feasible way to gauge any player's performance based on their specifications.

Submissions

The submission process is slightly longwinded but to ensure consistency between users, it unfortunately has to be so.

  • Login (through steam)
  • Enter hardware specs
  • Download and install Fraps (trial version)
  • Set up benchmark key (F11 default)
  • Set benchmark to record 5 minutes
  • Set benchmark to only record FPS
  • Record benchmark and write down which benchmark is on what map
    • one per game
    • try to record when near the center of the map, or in intense action. Recording the briefing screen or the initial rollout is not very helpful to gauge performance
    • only if >75 players are present
  • Submit benchmark(s)
  • Wait for approval from the admin to see your scores show up in the global stats
